1. Install typo3_blog extension from TER

2. Set the doktype ID for page type blog (example 73) and press butten "Update" and clear cache

3. Create a new page in page tree and edit this page and select the page type "Blog" and save the page.
   A page with page type Blog is a CATEGORY PAGE (bloglist)
   A pages under a Blogpage with page type Standard is a POST PAGE (blog single).
